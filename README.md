## EduLens – Student Performance Intelligence System

### Overview
EduLens is a Python-powered data analysis project focused on exploring
student academic performance data and extracting meaningful insights
through statistics and visualizations.

### Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib

### Key Capabilities
- Academic performance trend analysis
- Statistical summaries
- Insightful data visualizations

### Project Goal
To demonstrate practical data analysis skills using real-world
educational datasets.
